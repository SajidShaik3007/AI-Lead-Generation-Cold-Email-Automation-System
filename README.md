<h1>🚀 AI Lead Generation & Cold Email Automation System</h1>

<p>
This project is an end-to-end AI automation system that discovers potential business leads,
extracts company information, collects contact details, and generates personalized cold emails —
all in a fully automated workflow.
</p>

<hr>

<h2>🎯 Real-World Use Case</h2>

<div class="highlight">
Manually finding leads, researching companies, extracting emails, and writing cold emails is time-consuming.
This system automates the entire process — from discovering businesses in a target location
to sending personalized outreach emails — enabling scalable lead generation.
</div>

<hr>

<h2>🤖 Core Features</h2>

<ul>
  <li>📍 Lead discovery based on company type & location</li>
  <li>🌐 Website extraction for real businesses</li>
  <li>🧠 AI-based company analysis</li>
  <li>📧 Email extraction from websites</li>
  <li>📊 Automated lead storage in Google Sheets</li>
  <li>✉️ AI-generated personalized cold emails</li>
  <li>🚀 Automated email sending via SMTP</li>
</ul>

<hr>

<h2>🧠 Workflow Overview</h2>

<p>
1. User submits:
</p>

<ul>
  <li>Company type (e.g., Web Development Agency)</li>
  <li>Location (e.g., Mumbai)</li>
</ul>

<p>
2. AI agent searches and identifies relevant companies  
3. Extracts:
</p>

<ul>
  <li>Company Name</li>
  <li>Website URL</li>
</ul>

<p>
4. Stores initial lead data in Google Sheets  
5. Visits each website and extracts:
</p>

<ul>
  <li>Business details</li>
  <li>Services</li>
  <li>Contact email</li>
</ul>

<p>
6. Updates lead data with extracted emails  
7. AI generates a personalized cold email  
8. Sends outreach email automatically via SMTP  
</p>
